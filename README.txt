This is  a simple Store API application.
You can sort data: featured, company name, product name, sort (rating or price), fields, limit(data) and numericalFilters.
for example: /products?featured=true&company=caressa&sort=-rating,-price&fields=company,rating&limit=4&numericFilters=price>50,rating>=4

1.) git clone https://github.com/tutmaG/Store-API.git

2.) run: npm install or npm i 

3.) run: npm start
